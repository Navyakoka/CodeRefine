## Description
CodeRefine is a Generative AI-powered web application that analyzes source code to detect bugs, performance issues, security risks, and provides optimized code rewrites.

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI (Python)
- AI Model: Groq LLaMA
- API: REST API

## Features
- AI-based code review
- Bug detection
- Performance improvement suggestions
- Secure coding recommendations
- Optimized code rewrite

## How to Run the Project
1. Install dependencies:
pip install -r backend/requirements.txt

2. Start backend server:
uvicorn main:app --reload

3. Open frontend/index.html in browser

## Note
.env file is excluded for security reasons.
